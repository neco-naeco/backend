# 1. 온보딩 & 인증 (Onboarding & Authentication Flow)

### 목적

서비스 진입 및 최초 사용 준비

### 세부 플로우

- 회원가입
    1. 버튼 클릭
    2. id, 이메일, 비밀번호, 닉네임 입력
    3. 닉네임 중복 체크
    4. 버튼 클릭
- 로그인(로컬)
    1. 버튼클릭
    2. Id, password 입력
    3. 버튼클릭
- 첫 게임룸 생성 유도 (튜토리얼)
    1. AI Chat에서 AI 마스터가 프로젝트 컨셉(릴레이 게임)을 간략히 설명하고, 명령어 예시(예: "게임방 만들어줘")를 가이드.
    2. AI Chat에서 AI와 대화
    3. AI를 통해 게임룸생성

# 2. 게임룸 생성 & 환경 구성 (Game Room Setup Flow)

### 목적

채팅 명령을 통한 게임룸 생성 및 사용자 모집

### 세부 플로우

- 게임룸 생성 (방장)
    1. 메인화면 입장
    2. 방 소속 여부 확인
    3. AI마스터에게 방 생성 채팅으로 요청
        1. 게임 주제(난이도 포함) 리스트 조회
        2. 게임 주제 선택
        3. 확정 버튼 클릭
    4. AI 마스터가 방 정보 안내
- 유저 초대 (방장)
    1. 메인화면 입장
    2. 방 소속 여부 확인
    3. AI 마스터에게 유저 초대 채팅으로 요청
        1. 초대할 유저 닉네임 목록 전달
        2. AI 마스터가 초대 요청 성공 여부 안내
            1. 초대할 유저가 방에 소속되어 있으면 실패 안내 받음
    4. AI 마스터가 방 참가 인원 목록 안내
        1. 새 유저 입장 시 업데이트
        2. 해당 유저 거절 시 안내
- 초대 수락
    1. 메인화면 입장
    2. 방 소속 여부 확인
    3. AI 마스터가 브리핑
        1. 초대한 유저 닉네임 확인
    4. 수락하기 버튼 클릭
    5. 방 소속 여부 확인
    6. AI 마스터가 방 정보 안내
    7. AI 마스터가 방 참가 인원 목록 안내
        1. 새 유저 입장 시 업데이트
        2. 해당 유저 거절 시 안내
- 초대 거절
    1. 메인화면 입장
    2. 방 소속 여부 확인
    3. AI 마스터가 브리핑
        1. 초대한 유저 닉네임 확인
    4. 거절하기 버튼 클릭

# 3. 실시간 릴레이 엔진 (Relay Coding Flow)

### 목적

턴제 기반의 코드 릴레이 수행

### 세부 플로우

- 게임 시작 (방장)
    1. 시작하기 버튼 클릭
    2. 게임 화면 입장
    3. 모달로 미션 안내 (n 초 뒤 자동 사라짐)
- 게임 시작 (참여자)
    1. 게임 화면 입장
    2. 모달로 미션 안내 (n 초 뒤 자동 사라짐)
- 게임 진행(턴 진행자)
    1. 차례 안내
    2. 미션 진행도 확인
    3. 코딩
    4. 힌트보기 버튼 클릭
    5. 힌트 보고 코드 작성
    6. 제출 버튼 클릭
        - 성공: 다음 플레이어에게 턴 이동
        - 실패: 목숨 - 1, 이전 사용자가 망친 코드 상태 그대로 다음 플레이어에게 턴 이동
    7. AI 평가 대기
    8. 제한 시간 넘길 시
        - 성공: 다음 플레이어에게 턴 이동
        - 실패: 목숨 - 1, 이전 사용자가 망친 코드 상태 그대로 다음 플레이어에게 턴 이동
    9. 다른 유저 차례 시작
    10. 따라가기 버튼 클릭
- 게임 종료
    1. 마지막 유저가 제출 버튼 클릭
    2. AI 분석 대기
    3. 결과 확인
    4. 게임 종료 버튼 클릭
    5. 메인 화면 입장 (방 소속 x)
